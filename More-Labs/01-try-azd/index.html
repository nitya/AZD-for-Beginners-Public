    01: Try Azure Developer CLI¶

    In this hands-on lab, you will install the Azure Developer CLI (azd) on your development environment, authenticate with Azure, and explore core azd commands. You'll learn how to check your installation, view available templates, and understand the basic workflow of using azd as your "general contractor" for Azure deployments.
